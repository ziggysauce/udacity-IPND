Lesson 4

HTML Markup

The text content is what you see while the markup is how it's arranged. HTML also includes references to other documents (such as images) and links to other pages.

HTML Tags

An element makes up an entire tag and in general looks like this:

<tag name>content</tag name>

To create a bold tag we use:

<b>content</b> To create an emphasis tag (italicized) we use: <em>content</em>

HTML Attributes

Tags can have multiple attributes and use the following form:

<tag attribute="value">content</tag> Specifically an anchor tag is used for making links: <a>

Adding Images

To add an image we use the following code format:

<img src="url" alt="text"> Note that an image does not have a closing tag. This is known as a void tag (has no content in the tag).

Whitespace

Use either the line break code or the paragraph code to make a new line:

<br> or <p></p> Note that the br code is a void tag and doesn't require a closing tag. The difference between these two is that the line break is known to be "inline" while the paragraph is known to be a "block" element.

Span and Div

Span is an inline element while div is a block element. Both serve the same purpose and are used to create containers for their content.
